# 埃森梅莉亚 (Essenmelia)

> **“你的全能进度追踪与扩展枢纽”**

**Essenmelia** 是一个高度可定制、隐私安全且跨平台的个人记录与追踪工具。它不仅能帮助你记录追过的番剧、小说、电影和电视剧，更通过强大的插件化架构，允许开发者自由扩展功能与界面。

---

## 🌟 什么是 Essenmelia？

Essenmelia 最初诞生于一个简单的诉求：打造一款不受外界干涉、数据私有的离线追番进度追踪工具。

但我深知开发 Essenmelia 的初衷。纵观所有进度追踪工具，要么不支持封面展示，视觉割裂；要么进度藏得太深，无法在首页直观触达，甚至功能简陋到难以使用。好不容易遇到兼顾二者的软件，却往往深陷“伪离线”的泥潭——即便数据在本地，也强制用户联网，甚至以此为由捆绑 VIP 与订阅服务。

> **但这是必然的吗？“不提供在线资源，却强制在线使用”**

因此，Essenmelia 在开发之初就不局限于追番，为了兼容更多使用场景，它具有极高的视觉自由度：无论是隐藏封面、自定义卡片密度，还是特殊的书架布局，一切都为了打造最契合你的进度展示。

> **它是一个本地化、高定制、万物皆可记录的通用追踪器。无论是番剧电影，还是小说游戏——凡有进展，皆可留痕。Essenmelia 愿为你悉心收藏每一份热爱。**

## ✨ 相关功能

* **OCR 与视觉识别**（暂不支持 Windows ）
* 支持任意截图上传，点击 ✨ 即可自动识别原图中的封面、物体与文字。支持快速预览并一键转化为新事件。

* **高效步骤管理**
* 支持重命名、自由拖拽排序、多选及一键存档。
* 沉淀高频单步操作为“全局共享”，随时跨事件调用，将整套流程保存为模板，实现标准化任务的一键复用。
  
* **筛选与标签**
* 输入空格即刻化文字为标签，随心标记万物。
* 一键切换 全部 / 未开始 / 进行中 / 已完成。
* 支持“仅显示提醒”或“过滤无提醒任务”，屏蔽冗余，直击重点。

* **排序**
* 支持按时间、标题、进度、更新、步骤进行升降序排列。配合标签筛选，实现海量数据下的秒级定位。

* **批量操作**
* 长按触发多选模式，统一更改标签、状态与后缀。

* **提醒**
* 内置系统日历同步与应用原生通知两种提醒模式，确保进展不被遗忘。

> **就这？远远不止**

* **开放 API**（Beta）
* Deep Link 支持，通过 esml:// 或 essenmelia:// 协议，从网页、快捷指令一键唤起应用并触发操作。
* 支持通过 ADB 操作，为开发者自动化测试与极客脚本留出无限可能。

## 🧩 扩展

> **打破能力的边界**
> * Essenmelia 扩展通过 GitHub api 发现和安装扩展，它能将应用从一个单纯的“记录本”升华为个性化的**“全能媒体枢纽”**。

* 只需输入一个名字，扩展支持自动抓取并填充简介、封面、总集数（步骤）等所有信息。
* 支持在应用内播放视频、阅读小说和自动更新进度。
* 支持从 Bangumi 等开放平台导入记录和接入任意播放源。
