# 事件列表测试 (Event Test)

展示并操作系统中的事件，用于测试列表渲染、数据同步及权限管理。

## 功能
- **实时刷新**: 获取并显示最新的系统事件。
- **批量创建**: 一键生成模拟事件。
- **数据清理**: 确认后清空所有事件。

## 权限说明
- `readEvents`: 读取系统事件。
- `addEvents`: 添加新事件。
- `deleteEvents`: 删除事件。
- `systemInfo`: 获取系统信息。
