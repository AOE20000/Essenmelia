# API 测试工具 (API Tester)

测试 Essenmelia 扩展 API 的功能与响应。

## 功能
- **数据读取测试**: 调用 `getEvents` 获取事件总数。
- **UI 反馈测试**: 弹出 SnackBar 消息提醒。
- **网络连通测试**: 发起 HTTP GET 请求 (GitHub API)。

## 权限说明
- `readEvents`: 读取系统事件。
- `network`: 发起网络请求。
- `systemInfo`: 获取系统信息。
