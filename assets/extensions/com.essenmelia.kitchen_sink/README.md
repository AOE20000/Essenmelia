# 全能演示 (Kitchen Sink)

展示所有可用组件、API 指令与 JS 逻辑的参考扩展。

## 功能展示
- **基础输入**: TextField, Slider。
- **开关与选择**: Switch, SegmentedButton。
- **交互与反馈**: Button, ConfirmDialog, SnackBar。
- **媒体与进度**: Image, Badge, LinearProgress。

## 权限说明
- `readEvents`: 读取系统事件。
- `network`: 发起网络请求。
- `systemInfo`: 获取系统信息。
- `navigation`: 界面跳转。
