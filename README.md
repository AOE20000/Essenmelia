# 埃森梅莉亚 (Essenmelia)

欢迎来到*埃森梅莉亚 (Essenmelia)*一个记录与追踪你追过的故事、番剧、小说、电影、电视剧甚至事件的工具。

## 功能
- 导入和导出本地记录，轻松管理你的追踪清单
- 支持多种内容类型：事件、小说、番剧、电视剧、电影等
- 不作恶、跨平台
